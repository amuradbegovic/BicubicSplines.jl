# BicubicSplines.jl

BicubicSplines is a Julia implementation of Carl de Boor's bicubic spline interpolation algorithm. It was written in February 2026 by Adnan Muradbegović and Irhad Breščić for a college course. It now lives in this repo as a Julia package that's ready to use for anyone interested in this numerical method.

## Installation
```
using Pkg
Pkg.add(url="https://github.com/amuradbegovic/BicubicSplines.jl")
```
